# 執筆原稿ディレクトリ

## 運用ルール

各章の執筆完了後、必ず以下を行うこと：

1. `drafts/summaries/chXX_summary.md` に**章ごとの要約**（500字以内）を作成する
2. 要約には以下を含める：
   - 登場人物と行動
   - 物語上の重要な出来事・判明した事実
   - 伏線の設置/回収
   - 次章への引き継ぎ事項
3. 新たに登場した固有名詞は `style/glossary.md` に追記する
4. キャラクター間の関係変化があれば `backbone/relationships.md` を更新する

## なぜ要約が必要か

LLMのコンテキストウィンドウには上限があるため、過去の章の本文すべてを毎回参照することはできません。
**要約ファイルが「LLMの外部記憶」として機能**し、章をまたいだストーリーの一貫性を保ちます。
